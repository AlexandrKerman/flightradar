# FlightRadar
Простое консольное приложения для получения данных о воздушных судах и их последующей обработки.

Также поддерживается работа с файловой системой.

## Установка
1. Клонировать репозиторий
```bash
git clone https://github.com/AlexandrKerman/flightradar
```
2. Установить зависимости
```bash
poetry install 
```
3. Установить переменные окружения

## Запуск
- Запуск приложения с инструкцией
```bash
python main.py 
```
- Запуск приложения без инструкции
```bash
python main.py --skip-tutor 
```

## Тестирование
```bash
pytest
```
Сгенерировать отчёт о покрытии:
```bash
pytest --cov=src --cov-report=html
```
Общее покрытие кода на момент написания: 90%
